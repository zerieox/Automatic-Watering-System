# Automatic-Watering-System
Automatic Watering System
The video link is here https://youtu.be/B8TPR4UqWpw
List of components:
  1. Water pump
  2. Two water tubes
  3. Moisture sensor
  4. Nucleo-64 board
  5. 1k resistor
  6. transistor
  7. 9v Battery
